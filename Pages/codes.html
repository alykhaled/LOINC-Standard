LOINC Codes and Hierarchy

The LOINC standard is organized into a hierarchical structure that reflects the relationships between different types of observations. At the top level, LOINC is divided into five major axes:

  • Component: what is being measured or observed (e.g., glucose, hemoglobin)
  • Property: the characteristic of the component being measured (e.g., mass, concentration)
  • Time: when the observation was made (e.g., before breakfast, 2 hours postprandial)
  • System: the organ system or body part being observed (e.g., respiratory system, skin)
  • Scale: the scale or method used to make the observation (e.g., ordinal scale, visual analog scale)

Each LOINC code includes a unique combination of these five axes, which provides a comprehensive description of the observation. For example, the LOINC code for "Glucose, serum/plasma" is 2345-7, where:

  • Component = Glucose
  • Property = Mass
  • Time = Pt
  • System = Serum/Plasma
  • Scale = Qn

The hierarchical structure of LOINC allows for more precise and accurate identification of observations and facilitates data exchange between different healthcare information systems. It also enables users to search for related observations and navigate the LOINC code system more efficiently.

As of the latest release, LOINC 2.70, the standard includes over 95,000 unique codes organized into more than 92,000 classes. LOINC codes are available in both electronic and print formats and can be accessed through various software tools and platforms.
